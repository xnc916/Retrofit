网络请求讲解所需接口

GET请求

凡是没有写完整的url路径的地方，都是以https://api.github.com为URL端点。

GET /search/repositories

参数：

    q     
    string
    搜索关键字以及其它限定符|

    sort
    string
    排序字段，可以是stars，fortks或updated。默认：结果符合程度。

    order
    string
    排序顺序，可以是asc或desc，默认是desc。
例子

搜索流行的使用Java语言开发的库：

https://api.github.com/search/repositories?q=language:java&page=1

响应为：

POST请求

BASE_URL : http://admin.syfeicuiedu.com

请求地址：

/Handler/UserHandler.ashx?action=register

请求参数：

UserName      string    用户名
Password     string     密码

参数样例：

{
"Password":"654321"
"UserName":"qjd",
}

返回结果：

成功
{
"errcode": 1,                  //状态值
"errmsg": "注册成功！",        //返回信息
"tokenid": 171                 //用户令牌
}

已存在
{
"errcode":2,
"errmsg":"注册此用户名已存在！"
}
